## 🚀 Быстрый Старт для Разработчика

Этот проект имеет обширную и структурированную документацию. Чтобы быстро войти в курс дела и начать работу, следуйте этому пути:

1.  **Сначала поймите, КАК мы работаем.** Ознакомьтесь с нашим рабочим процессом, стандартами и точками согласования. Это — закон нашей команды.
    - **➡️ [Прочитать Рабочий Процесс](./04-process/01-workflow.md)**

2.  **Затем получите высокоуровневое представление о проекте.** Этот документ даст вам общее понимание архитектуры и технологий.
    - **➡️ [Прочитать Обзор Архитектуры](./02-architecture/01-system-overview.md)**

3.  **Погрузитесь в технические детали.** Архитектурные документы — это "библия" проекта. Стандарты кода обязательны к исполнению.
    - **➡️ [Изучить Технологический Стек](./02-architecture/02-tech-stack.md)**
    - **➡️ [Изучить Стандарты Качества Кода](./04-process/02-coding-standards.md)**
    - **➡️ [Изучить Руководство по Рефакторингу](./04-process/03-refactoring-guidelines.md)**

4.  **Посмотрите на текущие задачи.** Обзор досок покажет, что сейчас находится в разработке и как управлять задачами.
    - **➡️ [Открыть Обзор Канбан-Досок](./05-tracking/01-board-overview.md)**
