# Модели Данных Проекта: [Название Проекта]

_**Цель этого документа:** Описать структуру ключевых сущностей данных, с которыми работает приложение. Это помогает понять, из чего состоит "мир" нашего проекта и как его части связаны между собой._

---

## Модель 1: Пользователь (User)

_**Описание:** Представляет пользователя системы, его аутентификационные данные и персональную информацию._

| Поле          | Тип                 | Описание                                  | Пример                   |
|---------------|---------------------|-------------------------------------------|--------------------------|
| `id`          | `string` (UUID)     | Уникальный идентификатор пользователя.    | ` "a1b2c3d4-..." `       |
| `email`       | `string`            | Электронная почта, используется для входа. | ` "user@example.com" `   |
| `passwordHash`| `string`            | Хеш пароля.                               | ` "0x..." `              |
| `displayName` | `string` (optional) | Отображаемое имя пользователя.            | ` "John Doe" `           |
| `createdAt`   | `Date` (ISO String) | Дата и время создания учетной записи.     | ` "2023-10-27T10:00:00Z" `|
| `updatedAt`   | `Date` (ISO String) | Дата и время последнего обновления.       | ` "2023-10-27T12:30:00Z" `|

### Связи:
-   **Один ко многим** с моделью "Проект" (`Project`). Один пользователь может быть владельцем многих проектов.

---

## Модель 2: Проект (Project)

_**Описание:** Представляет собой основной рабочий объект в системе, к которому привязаны задачи._

| Поле          | Тип                 | Описание                                  | Пример                   |
|---------------|---------------------|-------------------------------------------|--------------------------|
| `id`          | `string` (UUID)     | Уникальный идентификатор проекта.         | ` "e5f6g7h8-..." `       |
| `ownerId`     | `string` (UUID)     | ID пользователя-владельца.                | ` "a1b2c3d4-..." `       |
| `name`        | `string`            | Название проекта.                         | ` "Релиз Q4" `           |
| `description` | `string` (optional) | Подробное описание проекта.               | ` "Запустить новую фичу..." ` |
| `status`      | `string` (enum)     | Текущий статус проекта.                   | ` "in_progress" `        |
| `createdAt`   | `Date` (ISO String) | Дата и время создания.                    | ` "2023-10-27T11:00:00Z" `|

### Связи:
-   **Многие к одному** с моделью "Пользователь" (`User`).
-   **Один ко многим** с моделью "Задача" (`Task`).

---

## _Инструкция по Добавлению Новых Моделей_

_Чтобы добавить новую модель, скопируйте и вставьте шаблон выше и заполните его для вашей сущности. Не забудьте описать ее связи с другими моделями._
