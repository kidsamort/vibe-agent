# Спецификации: Общие Утилиты

_**Цель этого документа:** Описать функциональность общих, переиспользуемых модулей и функций ("утилит"), которые не являются самостоятельными фичами, но используются в разных частях проекта._

---

## 1. Логгер (`/src/utils/logger.ts`)

_**Описание:** Централизованный модуль для логирования сообщений в консоль. Позволяет стандартизировать формат вывода и управлять уровнями логирования._

### API

#### `logger.log(message: string, ...args: any[])`
- **Назначение:** Вывод информационных сообщений.
- **Пример:** `logger.log('Пользователь успешно вошел', { userId: 123 });`

#### `logger.warn(message: string, ...args: any[])`
- **Назначение:** Вывод предупреждений о некритичных проблемах.
- **Пример:** `logger.warn('API ответил медленно', { responseTime: 500 });`

#### `logger.error(error: Error | string, ...args: any[])`
- **Назначение:** Вывод сообщений об ошибках.
- **Пример:** `logger.error(new Error('Не удалось сохранить данные'), { details: '...' });`

---

## 2. Форматер Дат (`/src/utils/date-formatter.ts`)

_**Описание:** Модуль для приведения объектов `Date` или ISO-строк к стандартизированным, читаемым форматам._

### API

#### `formatDate(date: Date | string): string`
- **Назначение:** Форматирует дату в локализованный, короткий формат.
- **Возвращает:** Строку вида `ДД.ММ.ГГГГ`.
- **Пример:** `formatDate(new Date()) // '27.10.2023'`

#### `formatDateTime(date: Date | string): string`
- **Назначение:** Форматирует дату и время.
- **Возвращает:** Строку вида `ДД.ММ.ГГГГ, ЧЧ:ММ`.
- **Пример:** `formatDateTime('2023-10-27T10:30:00Z') // '27.10.2023, 10:30'`
