# Пул Задач

Этот каталог предназначен для хранения папок с детальным описанием задач. Каждая подпапка должна соответствовать ID задачи (например, `DEV-101`) и содержать набор файлов, скопированных из шаблона `template/05-tracking/03-task-template/`.

Это позволяет централизованно хранить всю историю работы над задачами, даже если они больше не отображаются на активных досках.
