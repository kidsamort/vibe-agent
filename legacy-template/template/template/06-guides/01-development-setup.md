---

## 2. Первоначальная Настройка

1.  **Клонируйте репозиторий:**
    ```bash
    git clone [URL вашего репозитория]
    cd [название-папки-проекта]
    ```

2.  **Установите зависимости:**
    ```bash
    pnpm install
    ```

3.  **Настройте переменные окружения:**
    -   Скопируйте файл `.env.example` в `.env`:
        ```bash
        cp .env.example .env
        ```
    -   Откройте файл `.env` и заполните необходимые переменные, такие как `API_KEY`, `DATABASE_URL` и т.д. Обратитесь к [ссылке на документ с описанием переменных] за подробностями.

---

## 3. Запуск Проекта в Режиме Разработки

-   **Чтобы запустить frontend-приложение:**
    ```bash
    pnpm dev
    ```
    После этого приложение будет доступно по адресу `http://localhost:5173`.

-   **Чтобы запустить бэкенд (если есть):**
    ```bash
    pnpm start:backend
    ```

-   **Чтобы запустить все вместе (если настроено):**
    ```bash
    pnpm start:all
    ```

---

## 4. Проверка Установки

Чтобы убедиться, что все настроено правильно, выполните следующие шаги:

1.  **Запустите линтер:**
    ```bash
    pnpm lint
    ```
    Команда должна завершиться без ошибок.

2.  **Запустите тесты:**
    ```bash
    pnpm test
    ```
    Все тесты должны успешно пройти.

Если вы столкнулись с проблемами, обратитесь к разделу "Частые Проблемы" или к вашему тимлиду.
