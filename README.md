# Vibecoding Framework (vibe-agent) 🚀

**Vibe-Agent** — это CLI-фреймворк для эффективной совместной работы человека и ИИ-агентов по методологии Vibecoding.

## 🌟 Основная идея

Проект автоматизирует создание и ведение проектной документации, правил для ИИ и процессов разработки, превращая документацию в «живой» граф знаний.

## 🏗️ Статус проекта: Спринт 2 (Hydration & Intelligence)

Мы завершили базовый фундамент (Спринт 1) и перешли к реализации интеллектуального слоя:

- **Интервью с ИИ**: Генерация контекста на основе ответов пользователя.
- **MDX-Гидратация**: Автоматическое заполнение шаблонов данными.
- **Система плагинов**: Расширение возможностей CLI через модули.

## 🚦 Как запустить проект (Dev)

1. Установите [Bun](https://bun.sh/).
2. Склонируйте репозиторий.
3. Установите зависимости: `bun install`.
4. Запустите CLI в режиме разработки: `bun run src/index.ts`.
